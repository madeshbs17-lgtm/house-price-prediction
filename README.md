# 🏠 House Price Prediction using XGBoost

A machine learning project that predicts house prices using the **California Housing Dataset** and the **XGBoost Regressor**. This project covers data exploration, visualization, model training, and evaluation using regression metrics.

---

## 📖 Overview

The goal of this project is to build a regression model capable of predicting house prices based on housing characteristics from the California Housing dataset.

The project follows a standard machine learning workflow:

- Load the dataset
- Explore and analyze the data
- Visualize feature correlations
- Split the data into training and testing sets
- Train an XGBoost Regressor
- Evaluate model performance
- Visualize predictions

---

## 🚀 Features

- Exploratory Data Analysis (EDA)
- Correlation Heatmap
- Train-Test Split
- XGBoost Regression Model
- House Price Prediction
- Model Evaluation using:
  - R² Score
  - Mean Absolute Error (MAE)
